Application Overview:

An E-Commerce website for a shop based in India. It sells substances, products that is intended to prevent, destory, repell, mitigate any pest.
This website would be like other websites where I would include products which can be added to the users shopping cart. There will be multiple products on the website. Users can also give reviews on products. There will also be details about the product. It will include a shopping cart screen, a homepage with list of products, product details. There would be an Checkout options where user can complete their order with details. I might only include pick-up and not delivery option for users.
